Sorting Algorithms

1. Giới thiệu

Sorting là nhóm thuật toán dùng để sắp xếp dữ liệu theo thứ tự tăng hoặc giảm. Trong competitive programming, sorting là bước cơ bản trước khi áp dụng các kỹ thuật khác như greedy hoặc binary search.

2. Các thuật toán sắp xếp

  • Bubble Sort
  • Insertion Sort
  • Selection Sort
  • Merge Sort
  • Heap Sort
  • Quick Sort
  • Radix Sort

Bubble Sort

Bubble Sort hoạt động bằng cách liên tục so sánh hai phần tử kề nhau và hoán đổi nếu sai thứ tự. 


for(int i=0;i<n;i++)
for(int j=0;j<n-i-1;j++)
if(a[j] > a[j+1])
swap(a[j],a[j+1]);

Độ phức tạp: O(n²)

Insertion Sort

Insertion Sort chèn từng phần tử vào đúng vị trí trong phần đã sắp xếp. 


for(int i=1;i<n;i++){

int x=a[i];
int j=i-1;

while(j>=0 && a[j]>x){
a[j+1]=a[j];
j--;
}

a[j+1]=x;

}

Độ phức tạp: O(n²)

Selection Sort

Selection Sort mỗi lần tìm phần tử nhỏ nhất và đưa về đầu mảng. 


for(int i=0;i<n;i++){

int min=i;

for(int j=i+1;j<n;j++)
if(a[j]<a[min])
min=j;

swap(a[i],a[min]);

}

Độ phức tạp: O(n²)

Merge Sort

Merge Sort sử dụng chiến lược chia để trị. Mảng được chia đôi, sắp xếp hai nửa và sau đó trộn lại. 


void mergeSort(int l,int r){

if(l>=r) return;

int mid=(l+r)/2;

mergeSort(l,mid);
mergeSort(mid+1,r);

merge(l,mid,r);

}

Độ phức tạp: O(n log n)

Heap Sort

Heap Sort sử dụng cấu trúc dữ liệu heap để sắp xếp. 


make_heap(a,a+n);

for(int i=n-1;i>=0;i--){

swap(a[0],a[i]);
heapify(a,0,i);

}

Độ phức tạp: O(n log n)

Quick Sort

Quick Sort chọn một phần tử làm pivot và chia mảng thành hai phần. 


void quickSort(int l,int r){

int i=l,j=r;
int pivot=a[(l+r)/2];

while(i<=j){

while(a[i]<pivot) i++;
while(a[j]>pivot) j--;

if(i<=j){
swap(a[i],a[j]);
i++;
j--;
}

}

if(l<j) quickSort(l,j);
if(i<r) quickSort(i,r);

}

Độ phức tạp trung bình: O(n log n)

Radix Sort

Radix Sort sắp xếp các số theo từng chữ số từ thấp tới cao. 


for(int exp=1; max/exp>0; exp*=10)
countSort(a,n,exp);

Độ phức tạp: O(n * k)

3. Một số bài toán áp dụng

  • Sắp xếp điểm thi của học sinh
  • Sắp xếp đoạn thẳng theo tọa độ
  • Sắp xếp để áp dụng greedy
  • Sắp xếp trước khi dùng binary search
