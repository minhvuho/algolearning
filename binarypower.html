Lũy thừa nhị phân (Binary Exponentiation)

Mục lục

  • 1. Giới thiệu
  • 2. Ý tưởng của thuật toán
  • 3. Ví dụ minh họa
  • 4. Code C++
  • 5. Độ phức tạp
  • 6. Một số bài toán áp dụng

1. Giới thiệu

Lũy thừa nhị phân là thuật toán dùng để tính: 


a^n

một cách nhanh chóng.

Nếu tính trực tiếp bằng cách nhân n lần thì độ phức tạp là: 


O(n)

Trong khi đó, lũy thừa nhị phân chỉ cần: 


O(log n)

Thuật toán này được sử dụng rất nhiều trong competitive programming, đặc biệt khi tính lũy thừa với số mũ rất lớn.

2. Ý tưởng của thuật toán

Ý tưởng dựa trên biểu diễn nhị phân của số mũ.

Ví dụ: 


13 = 1101 (binary)

Ta có: 


a^13 = a^8 * a^4 * a^1

Trong quá trình tính toán, ta liên tục:

  • Bình phương cơ số
  • Giảm một nửa số mũ

Nếu bit cuối của n bằng 1 thì nhân kết quả với cơ số hiện tại.

3. Ví dụ minh họa

Tính: 


3^13

Các bước: 


13 = 1101

3^1
3^2
3^4
3^8

Sau đó nhân các lũy thừa tương ứng.

4. Code C++


long long binpow(long long a, long long n){

long long res = 1;

while(n > 0){

if(n % 2 == 1)
res = res * a;

a = a * a;

n /= 2;

}

return res;

}

Phiên bản có modulo (rất phổ biến trong CP): 


long long binpow(long long a, long long n, long long mod){

long long res = 1;

while(n > 0){

if(n % 2 == 1)
res = (res * a) % mod;

a = (a * a) % mod;

n /= 2;

}

return res;

}

5. Độ phức tạp

Thuật toán giảm một nửa số mũ sau mỗi bước. 


O(log n)

Do đó rất hiệu quả khi n rất lớn.

6. Một số bài toán áp dụng

  • Tính lũy thừa modulo
  • Tính nghịch đảo modulo
  • Thuật toán Fermat nhỏ
  • Tính Fibonacci bằng ma trận
  • Exponentiation trong số học

Lũy thừa nhị phân là một trong những thuật toán cơ bản nhưng cực kỳ quan trọng trong competitive programming.
