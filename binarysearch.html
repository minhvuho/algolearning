Tìm kiếm nhị phân

1. Giới thiệu

Binary Search là thuật toán tìm kiếm trên mảng đã được sắp xếp. Mỗi bước thuật toán chia đôi phạm vi tìm kiếm.

Độ phức tạp: 


O(log n)

2. Ví dụ

Tìm vị trí của số x trong mảng đã sắp xếp. 


int l=0,r=n-1;

while(l<=r){

int mid=(l+r)/2;

if(a[mid]==x) return mid;

if(a[mid]<x)
l=mid+1;

else
r=mid-1;

}

3. Một số bài toán áp dụng

  • Tìm phần tử trong mảng
  • Binary search trên đáp án
  • Tìm giá trị nhỏ nhất thỏa điều kiện
