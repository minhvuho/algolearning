Hash

1. Giới thiệu

Hash là kỹ thuật chuyển dữ liệu (thường là chuỗi) thành một số nguyên. Trong competitive programming, hash thường dùng để:

  • So sánh hai chuỗi nhanh
  • Tìm chuỗi con giống nhau
  • Phát hiện pattern trong string

Thay vì so sánh từng ký tự O(n), hash cho phép so sánh trong O(1).

2. String Hash

Ý tưởng cơ bản là biểu diễn chuỗi như một đa thức. 


hash(s) =
s[0]*p^0 +
s[1]*p^1 +
s[2]*p^2
...

Trong đó:

  • p là một số nguyên tố (ví dụ 31 hoặc 53)
  • mod là số lớn để tránh overflow

3. Rolling Hash

Rolling Hash cho phép tính hash của substring trong O(1).

Ta dùng prefix hash: 


H[i] = hash của prefix độ dài i

Hash substring: 


hash(l,r) = H[r] - H[l-1]*p^(r-l+1)

4. Collision

Collision xảy ra khi hai chuỗi khác nhau nhưng có cùng hash.

Để giảm xác suất collision:

  • Dùng mod lớn
  • Dùng double hash
  • Dùng base ngẫu nhiên

5. Ví dụ code

C++ implementation: 


const long long mod = 1000000007;
const long long base = 31;

long long power[1000005];
long long h[1000005];

void build(string s){

int n = s.size();

power[0] = 1;

for(int i=1;i<=n;i++)
power[i] = power[i-1]*base % mod;

for(int i=1;i<=n;i++)
h[i] = (h[i-1]*base + s[i-1]) % mod;

}

long long getHash(int l,int r){

return (h[r] - h[l-1]*power[r-l+1] % mod + mod) % mod;

}

6. Một số bài toán áp dụng

  • Kiểm tra hai substring có giống nhau không
  • Tìm pattern trong chuỗi
  • Longest Common Substring
  • Phát hiện palindrome
