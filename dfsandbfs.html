DFS và BFS

1. Giới thiệu

DFS (Depth First Search) và BFS (Breadth First Search) là hai thuật toán duyệt đồ thị cơ bản.

2. Ví dụ

DFS:


void dfs(int u){

visited[u]=true;

for(int v:adj[u])
if(!visited[v])
dfs(v);

}

BFS:


queue q;

q.push(s);

while(!q.empty()){

int u=q.front();
q.pop();

for(int v:adj[u])
if(!visited[v]){
visited[v]=true;
q.push(v);
}

}

3. Bài toán áp dụng

  • Connected components
  • Shortest path trong graph không trọng số
